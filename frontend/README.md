# Frontend - Crypto Calculator

Este directorio contiene el frontend en ReactJS que consume la API del backend NestJS.

## Ejecutar en local

1. Instalar dependencias:
    ```bash
    yarn install
    ```

````
2. Ejecutar el servidor de desarrollo:
   ```bash
yarn dev
````

## Variables de entorno

Usar `VITE_API_URL` para apuntar al backend, por ejemplo:

```bash
VITE_API_URL=http://localhost:3001
```
