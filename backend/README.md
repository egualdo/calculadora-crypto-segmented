# Backend - Crypto Calculator

Este directorio contiene el backend en NestJS que expone endpoints REST:

- `GET /api/rates`
- `GET /api/historical-rates`
- `POST /api/calculate`

## Ejecutar en local

1. Instalar dependencias:
    ```bash
    yarn install
    ```

````
2. Ejecutar el backend en desarrollo:
   ```bash
yarn start:dev
````

## Docker

El backend se ejecuta en el puerto `3001`.
